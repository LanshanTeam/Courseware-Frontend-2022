## 课程简介

> 重邮蓝山工作室是由重庆邮电大学信息化办公室指导的学生开发团队，包含了产品、前端、后端、UI、运维五个部门。为培养蓝山工作室 Web 前端方向的有生力量，蓝山前端研发部，编写本套配套课件，仅供学员预习、复习使用。
